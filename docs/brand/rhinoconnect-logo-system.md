# RhinoConnect Logo System

Brand assets live in `apps/mobile/assets/brand` and `apps/web/public/brand`.

- `rhinoconnect-mark.svg`: square app/profile/social mark using a one-horn rhino silhouette
- `rhinoconnect-logo-horizontal.svg`: web header and light-background wordmark
- `rhinoconnect-logo-light.svg`: splash/cover usage on deep blue
- Android launcher icons are generated into `apps/mobile/android/app/src/main/res/mipmap-*`
- Android native splash uses `apps/mobile/android/app/src/main/res/drawable/launch_logo.xml`
- Rhino icon direction references the CC0 Rhino SVG category on SVG Repo, then redraws the mark into RhinoConnect's own single-horn silhouette.

Primary colors:

- Rhino Blue: `#183F78`
- Pine Green: `#062B22`
- Lime CTA: `#DFFF25`
- Warm Cloud: `#F8F5EA`
- Connect Silver: `#B9BABD`
