## Product Direction

The current app is a working SaaS booking dashboard for tourism and service businesses. The marketplace version expands the product into a two-sided platform:

- Customers discover, compare, book, and pay for local services.
- Providers manage listings, schedules, bookings, payments, and customer communication.
- RhinoConnect operates the marketplace, trust layer, payments flow, and growth system.

The existing Next.js app should remain useful as the first provider/admin web console while the mobile marketplace and production backend are added step by step.
