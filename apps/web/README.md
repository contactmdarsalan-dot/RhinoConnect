# RhinoConnect Web

This is the current working Next.js app for RhinoConnect.

It includes:

- Public booking page
- Provider/admin dashboard
- Service CRUD with image/video galleries
- Bookings, payments, customers, availability, analytics, and settings
- Local JSON persistence for development

## Run From Repo Root

```bash
npm run dev
npm run lint
npm run build
```

## Run From This Folder

```bash
npm run dev
npm run lint
npm run build
```
