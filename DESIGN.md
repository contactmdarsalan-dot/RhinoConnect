# RhinoConnect Design System

## Scene

Travelers and local operators use RhinoConnect from phones in bright, busy places: airports, hotel lobbies, cafes, and outdoor travel stops. The interface should feel premium, legible, and steady under pressure.

## Visual Direction

- Product style: calm premium marketplace, not a decorative travel brochure
- Surface strategy: light, high-contrast, soft green-tinted neutrals
- Accent roles: pine for primary action, saffron for trust moments, sky for selected states
- Cards: used for repeated services, bookings, and account rows only
- Motion: short, functional transitions for navigation and state changes

## Mobile Rules

- Touch targets should be at least 44px
- Use safe areas for top bars, bottom nav, and fixed CTAs
- Use visible labels for form fields
- Keep one primary action per screen
- Do not rely on hover or color-only meaning
- Use Material icons consistently
- Support readable text at larger system font sizes

## Core Tokens

- Pine: `#006B5A`
- Alpine: `#102A2E`
- Cloud: `#F5F8F6`
- Mist: `#E8EFEB`
- Sky: `#B9E7F5`
- Saffron: `#F4B740`
- Coral: `#E95D4F`
- Ink: `#0B1B22`
- Slate: `#52646B`

## Typography

- Use system Roboto for Flutter Material compatibility
- Headings are bold and compact
- Body text uses 16px with generous line height
- Numeric prices should be visually prominent and stable
